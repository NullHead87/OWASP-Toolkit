About CTF Lab

Practice OWASP Security Toolkit tools in a safe offline environment.

How to use:
1. Open a challenge
2. Read the description and data
3. Use OWASP Toolkit tools to find the flag
4. Submit in format FLAG{...}
5. Check Writeups if stuck

Difficulty:
🟢 Easy — Basic tool usage (50-75 pts)
🟡 Medium — Combining techniques (150-200 pts)
🔴 Hard — Multi-step exploitation (300-350 pts)
🔵 Insane — Advanced chaining (500 pts)

Note: Progress is saved in memory only — reloading resets score.
